## [1.0.0] — 2025-05-07

### Added
- Initial release
- `SKILL.md` — core agent instructions with 5-step workflow
- Code traceability enforcement (mandatory file path + line range on all code references)
- Progressive template loading via `references/` directory
- **Requirements Analysis** template (`req-analysis.md`) — MoSCoW prioritization, use case diagrams, functional/non-functional requirements tables
- **Architecture Design** template (`arch-design.md`) — layered architecture diagram, tech stack rationale table, core design decision log
- **Database Design** template (`db-design.md`) — ER diagram, per-table field specifications, index strategy with justifications
- **Module Design + API Reference** template (`module-design.md`) — sequence diagrams, key function annotations, unified response format
- **Test Plan + Conclusion** template (`test-plan.md`) — unit/integration/system/performance test tables, summary and future work structure
- Bilingual README (English + Simplified Chinese)
